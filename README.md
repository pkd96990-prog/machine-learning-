<h1>Spam Email Detection using Logistic Regression</h1>

<p>
This project demonstrates a simple and effective <b>machine learning approach</b>
to classify messages as <b>spam</b> or <b>not spam</b> using
<b>Logistic Regression</b> and <b>Natural Language Processing (NLP)</b>.
</p>

<h3>Objective</h3>
<p>
To automatically detect spam emails by analyzing text content using
TF-IDF feature extraction and a supervised learning model.
</p>

<h3>Technologies Used</h3>
<ul>
    <li>Python</li>
    <li>NLTK</li>
    <li>Scikit-learn</li>
    <li>Pandas & NumPy</li>
    <li>Jupyter Notebook</li>
</ul>

<h3> Key Features</h3>
<ul>
    <li>Text preprocessing using NLP techniques</li>
    <li>TF-IDF based feature extraction</li>
    <li>Spam classification using Logistic Regression</li>
    <li>Model evaluation with accuracy and confusion matrix</li>
</ul>

<h3>Future Improvements</h3>
<ul>
    <li>Try Naive Bayes or SVM classifiers</li>
    <li>Use n-grams for better context understanding</li>
    <li>Deploy as a web app using Streamlit</li>
</ul>
